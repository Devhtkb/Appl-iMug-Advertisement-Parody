# Appl-iMug-Advertisement-Parody
An advertisement project about a mug with unnecessary features, the purpose of this project was to develop my skills in software and industry level production.

The project was made in two weeks, using a laptop with the following specs:
AMD Ryzen 7 5800H
NVIDIA Geforce RTX 3070 Laptop GPU
32GB DDR4 RAM

With these software used for the following purposes:
Blender: Modeling and UV, video editting, composing, captions
Gimp: Textures
Autodesk Maya 2023 & Arnold: Animating, rendering
